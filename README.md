## TMDB CLI Tool

Use TMDB API to fetch movie information and display it in the terminal. A solution to roadmap [assignment](https://roadmap.sh/projects/tmdb-cli)

### Usage:
```
tmdb-app --type "playing"
tmdb-app --type "popular"
tmdb-app --type "top"
tmdb-app --type "upcoming"
```
### TMBD api
Create an account on [TMDB](https://developer.themoviedb.org/reference/movie-now-playing-list) to get token and assign it to `token` in token.py file.
